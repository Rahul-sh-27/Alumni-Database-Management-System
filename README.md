#Alumni Management System

Alumni Management System is a web-based application built with PHP and MySQL for managing alumni records, events, announcements, and forum interactions. The system provides role-based access for Admin, Alumni, and Guests, enabling secure and efficient management of alumni-related operations.

#Technology Stack & Requirements

-Backend: PHP

-Database: MySQL

-Web Server: XAMPP (Apache + MySQL)

-Frontend: HTML, CSS, JavaScript

-Code Editor: VS Code, Sublime Text, Vim

-Browser: Chrome, Firefox

#Features
-Admin

--Secure Admin login and dashboard

--Add, update, delete announcements and events

--Manage forum posts: add, delete, reply

--Search alumni records efficiently

--Add new alumni and update financial records (donations, registration fees)

-Alumni

--Register and login securely

--Update personal profile

--Participate in forums: post, reply, interact

--Search fellow alumni

--View own donation history

-Guests

--View public events and announcements

--Search for alumni information

#Installation & Setup

Install XAMPP and start Apache and MySQL servers.

Copy the project folder Alumni Management System to C:\xampp\htdocs.

Open the browser and go to phpMyAdmin (http://localhost/phpmyadmin/).

Create a new database: we_miniproject.

Import the SQL file from the database folder into the database.

#Open the application in the browser:

-http://localhost/Alumni-Management-System/index.php
